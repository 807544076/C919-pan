## 设计

##### 上传文件：

**当前设计：**

网页端生成随机 key 和 iv，文件内容通过 AES.OFB 模式加密，同时从后端获取公钥，对 key 和 iv 进行加密，一起传输回后端。在后端用私钥解密 key 和 iv，再解密文件内容。

**下一步设计：**
* 后端文件独立aes密钥加密存储
* HMAC 与 rsa 签名实现
